这一节的任务是熟悉一些方便的STL容器，与python中的list、map等功能类似；可谓磨刀不误砍柴工，熟练使用各种容器可以让编写高难度算法时忽略繁杂的细节。

6.1 vector

使用vector<sometype> v；表示vector类型；需要头文件vector。
变长数组，仍可以用v[i]的形式访问里面的元素。
常用的几个方法是begin()、end()、size()、push_back()。

6.2 set

适合要求元素不能重复，或因元素过大或不易转化为整数而无法开哈希表的情况；可以做到自动去重并升序排序。

考CSP时一定要带一本STL工具书

6.3 string

string封装了一些字符串的功能，也保留了原来的字符数组式的访问功能；让字符串操作变得更加简洁了。
和vector类似可以在迭代器上加减来访问；

由于重载过运算符，所以可以用+=拼接字符串；用各种比较运算符比较字符串；还可以使用各种容器都有的erase()、size()、insert()、clear()等功能。
比较特别的还有substr()可以得到子串；replace()可以修改部分字符串；

6.4 map

建立一种基本类型到另一种基本类型的映射，比如之前的字符串hash，使用map会方便许多。
使用map<keytype, valuetype> mp声明mp，中间指明键类型和值类型。
可以用下标访问，赋值时只要当作一个没有边界的数组即可；也可以用迭代器访问，map会自动以键从小到大的顺序自动排列。

6.9 algorithm下的常用函数

函数 功能
max(float, float) 注意参数类型
min(float,float)
abs(int) 注意只能是整型
fabs(float) 注意这个函数在math头文件下面
swap() 交换变量的值
reverse(it,it2) 将数组指针或容器的迭代器在[it,it2)之间的元素反转顺序
next_permutation(it,it2) 修改为全排列中的下一个序列，到达最后一个会返回false
fill(it,it2,val) 将数组指针或容器的迭代器在[it,it2)之间的元素赋值为val
sort(it,it2,cmp) 喜闻乐见的排序
lower_bound(it,it2,val) 在[it,it2)之间寻找第一个>=val的位置，返回指针或迭代器
upper_bound(it,it2,val) 在[it,it2)之间寻找第一个>val的位置，返回指针或迭代器
